# Cue Audio Files

These files are used for lock-screen/background run cues via local notifications.

## Replace with premium voice
- Keep the same filenames.
- Keep each file under 30 seconds.
- Recommended format: `.wav` (16-bit PCM).
- Rebuild iOS app after replacing files.

Filenames currently used:
- `warmup_intro.wav`
- `warmup.wav`
- `run.wav`
- `walk.wav`
- `cooldown.wav`
- `summary.wav`
- `halfway.wav`
